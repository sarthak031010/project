# project
Modern, responsive product page built with HTML, CSS, and JavaScript. Features include interactive product gallery with zoom-on-hover, selectable color swatches, size options, tabbed product info, modals for size chart and color comparison, and scrollable carousels. Ideal for e-commerce portfolios and demo stores.


## Overview
This project is a **modern, responsive e-commerce product page** built using **HTML, CSS, and JavaScript**, designed to replicate a professional Shopify-style layout. The page is ideal for showcasing products online, portfolios, mock Shopify stores, or real e-commerce applications. It features a clean interface, interactive elements, and a fully responsive design that adapts seamlessly across desktop, tablet, and mobile screens.

---

## Features

### Navbar
- Responsive **navbar** with:
  - **Logo**
  - **Site name**
  - **Search bar**
  - **Shopping cart icon**
- Mobile-friendly layout with flexible wrapping and hover effects.
- Enhances navigation experience with intuitive and accessible design.

### Product Gallery
- **Main product image** with zoom-on-hover effect.
- Scrollable **thumbnail images** to switch the main display dynamically.
- **Active thumbnail** highlighting for better visual feedback.
- Supports multiple images to showcase product variations.

### Product Details
- Displays **product title, price, and description**.
- Interactive **color swatches** to select product colors.
- **Size dropdown** for choosing product size.
- “**Add to Cart**” button with hover effect for user engagement.
- **Compare Colors** feature to visualize multiple color options.

### Tabs
- **Tabbed navigation** for organizing product information:
  - Description
  - Product Information
  - Shipping Details
- Smooth toggling without page reload.
- Keeps the interface clean and organized.

### Modals
- **Size Chart Modal**: Helps customers select the right size.
- **Compare Colors Modal**: Shows all available color variants.
- Modals can be closed via:
  - Close button
  - Clicking outside
  - Pressing the **Escape key**

### Scrollable Carousels
- Horizontal scrollable **image boxes** for:
  - Related products
  - Bundles
  - Additional items
- Enhances user browsing experience and showcases product variety.

### Responsiveness
- Fully responsive design:
  - Desktop
  - Tablet
  - Mobile
- Uses **Flexbox** for layout adjustments.
- Ensures smooth user experience on all devices.

### Interactive Elements
- Hover effects on images, buttons, and swatches.
- Active state highlights for thumbnails and selected colors.
- Enhances engagement and usability.

---

## Technologies Used
- **HTML5** for semantic structure
- **CSS3** (Flexbox, responsive design, hover effects)
- **JavaScript** for DOM manipulation, event handling, tabs, modals, and interactive UI
- **Font Awesome** icons for search and cart
